SEO (Search Egine Optimization)
É o conjunto de técnicas usadas, geralmente divididas entre tecnologia, conteúdo e autoridade, para alcançar bom
posicionamento de páginas de um site no Google e em outros buscadores, gerando tráfego orgânico.
CLI (Comand Line Interface)
Ela representa um programa funcionando através de linhas de comando, que aceita entrada de texto para executar
funções do sistema operacional.
IDE (Integrated Development Environment)
É um software que auxilia no desenvolvimento de aplicações, sendo uma forma de criar aplicações de maneira mais
rápida, auxiliando em todo o processo de desenvolvimento de softwares, provendo diversos benefícios, como a análise
de todo o código a ser escrito para identificar bugs causados por um erro de digitação, autocompletam trechos de
códigos, e etc.
CRUD (Creat/Read/Update/Delete)
A sigla CRUD significa as iniciais das operações create (criação), read (leitura), update (atualização) e delete
(exclusão). Essas quatro siglas tratam a respeito das operações executadas em bancos de dados relacional (SQL) e
não-relacional (NoSQL). Essas operações pertencem ao agrupamento chamado de Data Manipulation Language (DML),
utilizado na linguagem Structured Query Language (SQL).
DNS (Domain Name System)
Converte nomes de domínio legíveis por humanos em endereços IP legíveis por máquina (por exemplo, 192.0.2.44).
IP (Internet Protocol)
É uma identificação digital que permite encontrar uma máquina na rede, como uma espécie de CPF do dispositivo.
Essa identificação é representada por um conjunto de quatro números, que podem variar de 0 a 225, a exemplo de
192.157.3.45, 0.0.0.0 ou 255.255.255.255.
API (Application Programming Interface)
Um conjunto de normas que possibilita a comunicação entre plataformas através de uma série de padrões e protocolos.
JSON (JavaScript Object Notation Object)
Formato leve de troca de informações/dados entre sistemas
PWA (Progressive Web Apps)
Um PWA é uma aplicação híbrida entre web e mobile.O site apresenta toda interface do navegador como barra de endereço,
botões, favoritos, etc, removidos. Como um site com aparencia de app.
WWW (World Wide Web)
Trata-se de um sistema interligado de arquivos e informações executados na internet. O modelo também é um sistema
de hipermídia que é capaz de conectar todo o planeta e transmitir informações para qualquer lugar.
HTTP (Hiper Text Transfer Protocol)
HTTP é um protocolo de transferência que possibilita que as pessoas que inserem a URL do seu site na Web possam
Esse sistema é a base da comunicação que existe em toda a Internet em que os sites e conteúdos que tragam hiperlinks
possam ser encontrados mais facilmente pelo públicover os conteúdos e dados que nele existem.
CMS (Content Management System)
Uma plataforma que permite criar, editar, publicar e remover conteúdos em sites de forma simplificada e a partir de
uma linguagem intuitiva.Uma alternativa para pessoas que não têm conhecimentos em programação e precisam, portanto, de
um método de gestão fácil.
UI (User Interface)
Onde são feitas as interações entre pessoas e softwares ou aplicativos. Essa área facilita a navegação dos usuários
nas plataformas digitais, permitindo que eles consigam ser orientados através de recursos visuais para concluírem suas
ações com praticidade.
UX (User Experience)
É o que o usuário sente ao utilizar um produto, sistema ou serviço, se relacionando especificamente com a experiência
do consumidor com determinado produto (independente da experiência ser boa ou ruim).
DOM (Document Object Model)
É uma interface padronizada que permite que navegadores e scripts possam manipular o conteúdo de uma página web sem a
necessidade de realizar atualizações e upgrades.
MVC (Model/View/Controller)
É um padrão de design onde se especifica que um aplicativo se consiste em um modelo de dados(model), respostas visuais às
entradas(view), e informações de controle(controller), mas separando os interesses em camadas.
Regex (Regular Expossion)
São padrões utilizados para identificar determinadas combinações ou cadeias de caracteres em uma string.Por meio dela,
podemos validar a entrada de usuários ou encontrar alguma informação em logs, documentação ou saída de comando.
Rest (Representation State Fromsfer)
Estilo arquitetônico aplicado para fornecer padrões entre sistemas de computador na web, facilitando a comunicação
entre eles.
SDK (Software Development Kit)
Essa ferramenta consiste em um grupo de recursos de desenvolvimento e códigos com gravação prévia que são utilizados
pelos programadores para construir aplicativos destinados a uma plataforma operacional.
ES (ECMAScript)
Padronização criada para a linguagem JavaScript.
DOP/POO (Object Oriented Programing)
É um paradigma de programação baseado no conceito de "objetos", que podem conter dados na forma de atributos, e
códigos, na forma de métodos.
XML (Extensible Markup Language)
Uma linguagem de marcação com regras para formatar documentos de forma que eles sejam facilmente lidos tanto por
humanos quanto por máquinas. No Brasil, ficou popularizado por ser o tipo de arquivo mais utilizado na emissão de notas
fiscais digitais.
